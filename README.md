## Introduction

基于vben-admin，引入了plop和handlebars可以一键生成模板的demo
## Install and use

- Get the project code

```bash
git clone https://github.com/anncwb/vue-vben-admin.git
```

- Installation dependencies

```bash
cd vue-vben-admin

yarn install

```

- run

```bash
yarn serve
```

- build

```bash
yarn build
```
